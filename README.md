# CV Application

A simple React application that lets users enter and edit their personal details, education history, and work experience to generate a basic CV.

🔗 **Project assignment:**  
https://www.theodinproject.com/lessons/node-path-react-new-cv-application

## Features
- Editable fields for personal information (name, email, phone)
- Sections for education and work experience
- Add and delete multiple entries for education and experience
- Reusable components for inputs and lists

## Key Learnings
- Setting up a React project with Vite
- Structuring an app with reusable components
- Managing form and list state with `useState`
- Passing data and handlers between components via props
- Rendering dynamic lists with `.map()` and using keys
- Using conditional rendering for edit/display modes
